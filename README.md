# 🧮 Máy Bấm Số Đơn Giản (Simple Counter)

Một project JavaScript cơ bản mô phỏng máy bấm số với các chức năng:

- ➕ Tăng số lên 1
- ➖ Giảm số xuống 1
- 🔄 Đưa số về 0


---

## 🛠 Công Nghệ Sử Dụng

- HTML5
- JavaScript (DOM Manipulation)
- EventListener

---

## 💡 Cách Hoạt Động

1. Lấy phần tử bằng `getElementById`
2. Tạo biến lưu giá trị số
3. Dùng `addEventListener` để xử lý sự kiện click
4. Cập nhật `textContent` sau mỗi lần thay đổi
